# 🎮 Questify - 게임처럼 일상을 관리하는 퀘스트 시스템

** 사용자 기반 맞춤형 퀘스트 추천 • 실시간 활동 트래킹 • 노션 연동 자동화**

---

## ✨ 핵심 기능

- **🧩 MBTI 맞춤형 퀘스트**  
  사용자 성향에 딱 맞는 메인/서브 퀘스트 제공 (INTJ에게 어울리는 논리적 도전과제 등)

- **⏱️ 실시간 타이머 & 활동 로그**  
  시작/일시정지/완료 기능, 중간 휴식 시간 자동 기록

- **📅 자동화된 일일 마무리**  
  AI가 분석한 하루 요약 + 노션 타임라인 페이지 자동 생성

- **🔗 노션 연동**  
  퀘스트/활동 기록이 실시간으로 노션 DB와 동기화

---

## 🛠️ 기술 스택

| 영역 | 기술 |
|------|------|
| **프론트엔드** | React, Tailwind CSS, Shadcn/ui, Context API |
| **백엔드** | FastAPI, pydantic_ai (AI 에이전트), Uvicorn |
| **데이터베이스** | Notion API (캐릭터/퀘스트/활동로그/일기 DB) |
